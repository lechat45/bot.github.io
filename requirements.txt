# DÉPENDANCES EXTERNES (pip install -r requirements.txt) :
#
#   alpaca-trade-api >= 3.0.0
#       → SDK officiel Alpaca — utilisé exactement comme le code fourni :
#           tradeapi.REST(KEY, SECRET, URL, api_version='v2')
#           api.get_bars(symbol, TimeFrame.Hour, limit=N)
#           api.get_crypto_bars(symbol, TimeFrame.Hour) — pour BTC/USD, ETH/USD
#           api.submit_order(symbol, notional, side, type, time_in_force)
#           api.close_position(symbol) — liquidation stop-loss/take-profit
#           api.list_positions() — positions ouvertes et P&L
#           api.get_account() — equity, cash, buying_power
#       → Rôle : interface officielle broker — plus stable que les appels REST bruts
#
#   pandas-ta>=0.3.14b
#       → Calcul vectoriel de tous les indicateurs techniques sur DataFrame :
#           ta.hma(close, length=10)          — Hull MA vitesse (code fourni)
#           ta.ema(close, length=100)         — Tendance longue (code fourni)
#           ta.ema(close, length=9/21)        — Croisement court terme
#           ta.macd(close, fast=12, slow=26)  — Momentum MACD
#           ta.rsi(close, length=14)          — Sur-achat/sur-vente
#           ta.bbands(close, length=20, std=2)— Bollinger Bands
#           ta.adx(high, low, close, 14)      — Force de tendance
#           ta.atr(high, low, close, 14)      — Volatilité pour stops dynamiques
#           ta.sma(volume, length=20)         — Volume moyen pour spike detection
#       → Rôle : moteur de calcul technique — vectoriel, précis, maintenable
#
#   pandas >= 1.5.0
#       → DataFrame requis par pandas-ta et alpaca-trade-api
#       → Manipulation des séries OHLCV retournées par le SDK Alpaca
#       → Rôle : structure de données centrale pour l'analyse technique
#
#   numpy >= 1.24.0
#       → Dépendance de pandas et pandas-ta
#       → Calculs numériques vectoriels (score composite, ATR, etc.)
#       → Rôle : couche de calcul bas niveau
#
#   requests >= 2.31.0
#       → Appels HTTP vers Groq Cloud pour l'analyse de sentiment :
#           POST https://api.groq.com/openai/v1/chat/completions
#           Modèle : llama-3.3-70b-versatile (rapide, gratuit sur console.groq.com)
#       → Rôle : seul appel HTTP manuel — tout le reste passe par les SDK
#
